## How to add new code/features

**Via Terminal/VS Code**
---

```bash
git branch `feature-name` -c
git add .
git commit -m "commit message"
git push origin `feature-name`
```

- Give your feature a name that describes what it does/is supposed to do (doesn't have to be long, just descriptive enough)
- Wait for pull request to be merged or rejected based on code quality and functionality

**Via GitHub Desktop**
---

- Create a new branch
- Add and commit changes
- Push changes to remote repository
- Create pull request

**TEAM LEADER/PERSON IN CHARGE OF CODE REVIEW WILL MERGE CODE, NO ONE ELSE TO AVOID DISCREPANCIES**

- Please make sure to branch your code as to not affect the main branch
- Ensure your commits are clear and descriptive as well
- Create push requests for review each time you make changes
